Using @keyframes we can animate elements. This typically applies as soon as the element is loaded.

Triggering animations repeatedly

If we animate based on a class, we can trigger the animation on demand. For example, by toggling the class on the click event.

If we want to trigger the animation again, we can remove the class on the animationend event.

Once the animation is finished, clicking the section will restart the animation.

Infinite and alternate

Setting the animation-iteration-count to infinite we can animate elements infinitely.

Setting the animation-direction to alternate we can animate elements back and forth.
